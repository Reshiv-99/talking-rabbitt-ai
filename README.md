# Talking Rabbitt – AI Data Assistant

Talking Rabbitt is an AI-powered conversational analytics tool.

Users can upload a CSV dataset and ask questions in natural language.  
The system provides insights from the data.

## Features
- Upload CSV dataset
- Ask questions about the data
- AI-generated insights
- Simple web interface

## Tech Stack
- Python
- Streamlit
- Pandas

## How to Run

1. Install dependencies
pip install -r requirements.txt

2. Run the app
streamlit run app.py
